# AGRO.CHIKULAY | CASE IH ANALYTICS CENTER

## 📋 Опис проекту
**agro.chikulay** — це інноваційний інтелектуальний дашборд для моніторингу агросектору в режимі реального часу. Система об'єднує глобальні ринкові дані, локальні новини України, AI-аналітику та телеметрію парку техніки Case IH у єдиному високотехнологічному інтерфейсі.

Проект розроблений з акцентом на преміальну естетику (Dark Mode / Glassmorphism) та максимальну функціональність як на десктопних моніторах, так і на мобільних пристроях.

---

## ⚙️ Основна механіка та функціонал

### 1. Інтелектуальний інтерфейс (Smart Layout)
*   **Динамічна сітка (Dynamic Grid):** Використовує 4-колонкову структуру, яка автоматично підлаштовується під обсяг контенту.
*   **Вирівнювання по висоті:** Спеціальний алгоритм CSS Grid вирівнює всі інформаційні блоки по висоті, забезпечуючи ідеальну геометричну чіткість.
*   **Мобільна адаптація:** Повна трансформація в лінійну стрічку на смартфонах для зручного перегляду "на ходу".

### 2. Аналітичний центр та AI Brain
*   **AI Коментатор:** Система генерує оперативні рекомендації на основі актуальних новин (наприклад, поради щодо закупівлі добрив або вікон для експорту).
*   **Live-бігучий рядок:** Відображає актуальні ціни на світових біржах (CBOT, Matif) та внутрішні ціни України (Елеватори, Порти).
*   **Інфографіка:** Вбудовані візуальні індикатори експорту зерна, запасів в елеваторах та логістичних потужностей (Дунай, Залізниця).

### 3. Телеметрія флоту Case IH (FieldOps Integration)
*   **Розширений моніторинг:** Відображення ключових моделей (Steiger Quadtrac 715, Magnum, Optum, Puma, Axial-Flow).
*   **Real-time показники:** Рівень пального (Fuel), швидкість (Speed), продуктивність/навантаження (Load) та агрономічні параметри.
*   **Технічна документація:** Картки містять детальний опис технологій (трансмісії CVXDrive, двигуни FPT, системи Harvest Command).

### 4. Точне землеробство (Precision Farming)
*   **NDVI Моніторинг:** Візуалізація вегетаційного індексу Хмельницької області на основі супутникових знімків.
*   **Soil & Weather Sensors:** Дані з польових датчиків про вологість ґрунту, температуру та швидкість вітру.

## 🛠 Технічні налаштування

### Файлова структура:
*   `index.html`: Семантична структура з використанням HTML5 та SVG-іконок.
*   `index.css`: Комплексна система стилів з використанням CSS Variables для легкого брендування та Media Queries для адаптивності.
*   `app.js`: Ядро системи. Обробляє логіку оновлення таймера (60-секундний цикл), рендеринг карток техніки та випадкову генерацію AI-інсайтів.

### Як налаштувати / змінити дані:
1.  **Зміна цін:** Відредагуйте масив `tickerData` у файлі `app.js`.
2.  **Додавання техніки:** Достатньо додати новий об'єкт у масив `caseFleetData` — система автоматично згенерує нову картку та впише її в сітку.
3.  **Зміна регіону NDVI:** Замініть `ndvi_map_ukraine.png` на інше зображення та оновіть текст у `index.html`.
